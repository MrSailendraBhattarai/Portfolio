# Sailendra Bhattarai — Portfolio

A personal portfolio site built with **React + Vite**.

## Stack
- React 19
- Vite 8 (dev server on port 5000)
- Lucide React (icons)
- Plain CSS (no Tailwind/UI library)

## Running the app
```
npm run dev
```
The dev workflow `Start application` handles this automatically. Hot-module reload is enabled.

## Build
```
npm run build
```
Output goes to `dist/`.

## Project structure
- `src/App.jsx` — main single-file app (all sections: hero, about, timeline, skills, projects, gallery, certificates, contact)
- `src/components/ProjectCard.jsx` — reusable project card component
- `src/App.css` / `src/index.css` — all styles and CSS variables
- `public/` — images, icons, CV PDF
